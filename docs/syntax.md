# Tuelang Syntax

## Variables
Define variables using `$`.

`$ myvar = "Hello World!";`

End all statements in a semicolon.

## Input
Use `cue` for user input.

`$ input = cue "Enter some stuff";`

## Output
Use `echo` for output.

```
$ output = "Text";
echo output;
```

## Conditionals

```
if (condition) {
   ...
} elseif (condition) {
   ...
} else {
   ...
}
```

## Comments

```
~~ comment
```
